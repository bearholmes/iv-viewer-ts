Quick Start

1. Install

npm install iv-viewer-ts

2. Import

import ImageViewer from 'iv-viewer-ts';
import 'iv-viewer-ts/dist/iv-viewer-ts.css';

3. Use

const viewer = new ImageViewer('#container');
viewer.load('image.jpg', 'high-res.jpg');
